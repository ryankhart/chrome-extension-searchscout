# Privacy Policy for SearchScout

**Effective Date:** December 25, 2025

## Overview

SearchScout is a Chrome extension that adds a right-click context menu to search selected text across multiple customizable search engines. This privacy policy explains how SearchScout handles user data.

## Data Collection

**SearchScout does not collect, store, or transmit any personal data.**

The extension operates entirely locally on your device and does not communicate with any external servers.

## Data Storage

SearchScout stores the following data locally on your device using Chrome's Storage Sync API:

- **Search engine preferences**: Which search engines you have enabled or disabled
- **Custom search sites**: Any custom search engines you have added
- **Display order**: The order in which search engines appear in your context menu

This data is stored using Chrome's built-in sync storage, which means it may sync across your Chrome browsers if you're signed into Chrome with sync enabled. This syncing is handled entirely by Google Chrome and is subject to Google's privacy policies.

## Data Sharing

SearchScout does not:
- Collect any personally identifiable information
- Track your browsing history
- Monitor your search queries
- Share any data with third parties
- Sell or transfer any user data
- Use analytics or tracking services

## Permissions

SearchScout requests the following permissions:

- **contextMenus**: Required to create the right-click context menu items
- **storage**: Required to save your search engine preferences locally

## Changes to This Policy

If we make changes to this privacy policy, we will update the effective date at the top of this document.
